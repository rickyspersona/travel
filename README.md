# Travel Reports

兩個 sub-folder 各自存放 HTML 旅遊/出差報告:

```
travel/
├── family-trip/    ← 家庭旅遊(沖繩 5 日、員工旅遊等)
│   └── demo.html
└── business-trip/  ← 商務出差
    └── demo.html
```

## Vercel 部署

1. 登入 https://vercel.com → New Project → Import `rickyspersona/travel`
2. **不用改任何設定**(Framework Preset: Other,Root Directory: `./`)
3. Deploy 後預設路徑:
   - `https://travel-xxx.vercel.app/family-trip/demo.html`
   - `https://travel-xxx.vercel.app/business-trip/demo.html`

## 之後新增報告

直接 commit/push 一個新的 `.html` 到對應資料夾,Vercel 會自動重新部署。
